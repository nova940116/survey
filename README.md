## nova's survey platform

---

This is simple survey platform. you can submit survey and see survey result.

This service only supports Korean.

Currently in progress survey | https://survey.novauniverse.me/chicken  
All Survey Lists | https://survey.novauniverse.me

### Project Stacks

<b>Frontend</b>

- Typescript
- Next.js
- Tailwind CSS

<b>Backend</b>

- Node.js
- AWS DynamoDB
- AWS Lambda
- AWS API Gateway
- Serverless
